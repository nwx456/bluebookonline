If you're taking AP U.S. History, AP English Language, AP Psychology, or any of the **16 fully digital AP exams**, your free-response answers aren't handwritten — they're **typed in Bluebook**. That changes pacing, formatting, and how graders read your work.

This guide covers rubric-aligned typing strategies for every major FRQ type, plus how to practice on [AP Practice Exam Online's FRQ module](/dashboard/upload?kind=frq).

## Which Exams Require Typed FRQs?

Fully digital exams (see [complete 2026 list](/blog/fully-digital-vs-hybrid-ap-exams)) include:

- **History:** APUSH, World, European, Human Geography
- **English:** AP Lang (3 essays), AP Lit (3 essays)
- **Social science:** AP Psych, AP Gov, AP Comp Gov
- **STEM:** AP CSA (Java code), AP Environmental Science
- **Other:** AP Art History, AP Latin, AP Seminar

Hybrid exams (Calc, Bio, Chem, Physics, Stats, Econ) still use **paper FRQ booklets** — see [hybrid booklet guide](/blog/hybrid-ap-exam-frq-booklets).

## Bluebook FRQ Tools You Should Know

Before exam day, practice with these Bluebook features (detailed in our [interface guide](/blog/bluebook-interface-explained)):

- **Rich text editor** for essays (bold/italic limited — don't rely on formatting for points)
- **Code editor** for AP CSA with Java Quick Reference panel
- **Split screen** — prompt on one side, response area on the other
- **Timer** — visible countdown per section; no going back to previous FRQs in some formats

## Strategy by FRQ Type

### Short-Answer Questions (APUSH, World, Euro)

**Time:** ~12 minutes per SAQ (3 questions, ~40 minutes total)

**Structure that earns points:**
1. **Answer the specific part** (a, b, c) — label if the interface allows
2. **One concrete piece of evidence** per point
3. **2–4 sentences** per part — graders scan for rubric keywords

**Typing tip:** Outline in 30 seconds mentally, then type continuously. Don't over-edit — you lose time.

### DBQ (Document-Based Question)

**Time:** ~60 minutes (APUSH/World/Euro)

**Recommended time split:**
| Phase | Minutes | Action |
|---|---|---|
| Read docs + plan | 15 | Group documents, write thesis |
| Draft | 35 | 4–5 body paragraphs with doc references |
| Review | 10 | Check thesis, add doc if missing |

**Typing tip:** Type your thesis first, then build backward. Use "(Doc 3)" inline — no formal citation needed.

### LEQ (Long Essay Question)

**Time:** ~40 minutes

**Structure:** Thesis → 2–3 body paragraphs with specific evidence → brief conclusion

**Typing tip:** Aim for **3–4 paragraphs minimum**. A common 5-point LEQ needs contextualization, thesis, evidence, analysis, and complexity.

### AP English Language Essays

Three essays, ~40 minutes each:

1. **Synthesis** — cite 3+ sources; tag sources as you read
2. **Rhetorical analysis** — quote specific devices from the passage
3. **Argument** — clear claim + developed evidence

**Typing tip:** Leave 3 minutes to re-read your thesis and fix obvious typos manually (no spell check).

### AP Computer Science A (Java Code)

Four FRQs typed in a code editor with Java Quick Reference.

**Strategy:**
- Read all 4 prompts in the first 5 minutes
- Start with your strongest question type (often Methods or ArrayList)
- Write compilable code — partial credit requires syntactically valid attempts
- Use meaningful variable names; comment only if it clarifies logic

Practice Java FRQs via [FRQ upload](/dashboard/upload?kind=frq) with AI rubric feedback.

### AP Psychology FRQs

Two FRQs, ~25 minutes each. Apply concepts to scenarios with **defined terms + applied example**.

**Typing tip:** Bold isn't available for emphasis — use clear topic sentences instead.

## Universal Typing Rules for Exam Day

1. **No spell check** — practice without autocorrect now
2. **Answer every part** — blank = zero; partial > blank
3. **Use all allotted time** — review beats submitting early
4. **Don't delete entire responses** — edit in place when possible
5. **Watch the timer** — Bluebook shows section countdown
