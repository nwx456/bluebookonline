Choosing between AP Computer Science A and AP Computer Science Principles? They sound similar but test **very different skills**.

## Quick Comparison

| Feature | AP CSA | AP CSP |
|---|---|---|
| Focus | Java programming | Computing concepts broadly |
| MCQs | 40 (50%) | 70 (70%) |
| FRQ/Project | 4 Java coding FRQs | Create Performance Task (30%) |
| Format | Fully digital | Fully digital |
| Best for | Future CS majors | First computing course |
| Pass rate (2025) | ~67% | ~64% |

## AP Computer Science A

- Write **Java code**: classes, arrays, ArrayList, 2D arrays, inheritance
- FRQs require working programs typed in Bluebook
- Heavier on Units 3–4 (conditionals, iteration) and Unit 6 (arrays)

[CSA complete guide](/blog/ap-csa-complete-guide) · [Score calculator](/tools/ap-score-calculator/ap-csa)

## AP Computer Science Principles

- Covers **big ideas**: data, algorithms, internet, impact of computing
- **Create PT** (due April 30) = 30% of score — a programming project + video
- MCQs test concepts, not language-specific syntax

[CSP complete guide](/blog/ap-csp-complete-guide) · [Score calculator](/tools/ap-score-calculator/ap-csp)

## Which Should You Take?

**Take CSP if:** You've never coded, want a broad intro, or are exploring CS.

**Take CSA if:** You know Java (or learn quickly), want programming depth, or plan a CS major.

**Take both if:** Your schedule allows — CSP in one year, CSA the next.
