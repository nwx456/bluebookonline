The Bluebook app is the **only** way to experience the real Digital SAT interface before test day. If you have only practiced on paper or third-party sites, you are missing tools that save seconds on every question.

This guide walks through every Bluebook feature specific to the SAT — including Desmos, grid-in entry, and adaptive module transitions.

> **Related:** [Complete guide](/blog/digital-sat-complete-guide) · [Desmos tips](/blog/desmos-sat-math-tips) · [Grid-in guide](/blog/sat-grid-in-student-produced-response-guide)

## The Header Bar

At the top of every screen:

- **Section indicator** — shows "Reading and Writing" or "Math" and current module (1 or 2)
- **Directions dropdown** — re-read section instructions without losing your place
- **Timer with Hide** — countdown visible by default; hide it if it causes anxiety (unhide for final pacing)
- **Annotate** — highlight text and attach short notes to passages (R&W only)
- **Calculator** — opens Desmos graphing calculator (Math only, available for every question)
- **Reference** — formula reference sheet for Math (geometry formulas, etc.)

## Reading & Writing Interface

### Passage Layout
Each question shows a **short passage** (25–150 words) with one question. There is no long-passage format — one question per text.

### Key Tools for R&W

| Tool | How to use | When to use |
|---|---|---|
| **Highlight** | Select text → click Highlight | Mark key sentences in inference questions |
| **Annotate** | Select text → Add Note | Jot a 3-word reminder ("author skeptical") |
| **Eliminate** | Click X on answer choices | Cross out wrong options on grammar questions |
| **Mark for Review** | Click flag icon | Any question you are not 80% sure about |

**Tip:** Annotations do not transfer between questions. Use them only when you plan to return via mark-for-review.

### Module Transition
When Module 1 ends, Bluebook shows a **break screen** (not the 10-minute section break — that comes after all of R&W). You cannot return to Module 1 questions after starting Module 2.

## Math Interface

### Question Types
- **Multiple-choice** — click to select; change anytime before submitting the module
- **Student-produced response (grid-in)** — type your answer in the entry field (up to 5 characters)

### Desmos Calculator

Click **Calculator** in the header to open Desmos. It appears as a resizable panel alongside the question.

High-value uses:
- Graph systems of equations → read intersection point
- Find quadratic roots and vertex
- Regression on scatterplot data

Full tactics: [Desmos SAT Math tips](/blog/desmos-sat-math-tips).

### Grid-In Entry

For student-produced response questions:
1. Click the answer field below the question
2. Type your answer (number, decimal, or fraction)
3. Up to **5 characters** — see [grid-in rules](/blog/sat-grid-in-student-produced-response-guide)
4. No negative signs accepted

## The Footer Bar

- **Question X of Y** — click to open the question grid
- **Back / Next** — navigate one question at a time
- **Submit Module** — appears on the last question of each module

## The Question Grid

Click "Question X of Y" to see all questions in the current module:

- **Filled square** — answered
- **Empty square** — not yet answered
- **Flag icon** — marked for review

Use the grid in the final 3–5 minutes of each module to:
1. Return to flagged questions
2. Fill in any blanks (no guessing penalty)
3. Verify you did not accidentally skip a question

## The 10-Minute Break

After Reading & Writing (both modules), you get a **10-minute break** before Math begins.

- Leave the room if your center allows
- **Phone must stay off**
- Snacks and water are fine
- You cannot review notes or discuss questions

See the full [exam day checklist](/blog/digital-sat-exam-day-checklist).

## Adaptive Module 2 Transition

After Module 1, Bluebook loads Module 2 automatically. You may notice:
- Questions feel harder or easier depending on your Module 1 performance
- There is no indicator telling you which path you received

Understanding routing: [adaptive testing guide](/blog/how-digital-sat-adaptive-testing-works).

## Practice the Interface Before Test Day

1. Download Bluebook and complete at least one full practice test
2. Deliberately use every tool (highlight, Desmos, mark-for-review, grid-in)
3. Practice additional tests on [AP Practice Exam Online](/exams?program=sat) in Bluebook-style layout
4. Review [Module 1 strategy](/blog/digital-sat-module-1-strategy) so tool usage does not slow you down

The interface should feel boringly familiar on test day — that is the goal.
