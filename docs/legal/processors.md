# Data processors (sub-processors)

AP Practice Exam Online uses the following third-party services to process personal data:

| Processor | Purpose | Data processed |
|-----------|---------|----------------|
| Supabase | Database, authentication, file storage | Account, uploads, attempts |
| Google Gemini | AI question extraction and grading | PDF content, question text |
| Anthropic Claude | AI question extraction (AP) | PDF content, question text |
| Google AdSense | Advertising (with consent) | Cookie identifiers, usage |
| Email provider (Resend/SMTP/Gmail) | Transactional & marketing email | Email address, username |

Cross-border transfers may occur. Users are informed at signup based on their declared country/region.
